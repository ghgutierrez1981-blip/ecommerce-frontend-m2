# 🛒 E-commerce Tarot del Sol 19

## 📌 Descripción
Este proyecto corresponde al desarrollo de un e-commerce frontend enfocado en la venta de servicios de orientación espiritual, como lecturas de tarot y sesiones personalizadas.

## 🚀 Tecnologías utilizadas
- HTML5 semántico
- Bootstrap 5
- JavaScript
- jQuery
- Git y GitHub

## 🧩 Funcionalidades
- Visualización de productos en formato card
- Página de detalle de producto
- Carrito de compras dinámico
- Contador de productos en navbar
- Botón “Ir arriba”
- Diseño responsivo

## 💻 Cómo ejecutar
1. Clonar el repositorio
2. Abrir `index.html` en el navegador
